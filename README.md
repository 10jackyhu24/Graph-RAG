# Graph RAG

## install
- [python 3.11](https://www.python.org/downloads/)
- [ollama](https://ollama.com/) (model: qwen3:8b)
- [nodejs](https://nodejs.org/zh-tw/download)

```
cd backend
pip install -r requirements.txt
```

## run
**Two terminals are required.**

**Frontend:**
```
cd frontend
npm run dev
```

**Backend**
```
cd backend
python main.py
```
